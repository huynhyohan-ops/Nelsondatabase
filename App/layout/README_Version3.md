# Nelsondatabase - Local Demo (Teal Theme)

Phiên bản này là demo local (HTML/CSS/JS) cập nhật theo tông xanh/teal giống ảnh mẫu anh gửi: giao diện tối, nền gradient, cards màu teal đậm, điểm nhấn màu cam cho cảnh báo.

Cấu trúc file
- index.html — trang chính demo
- styles.css — theme + layout styles (teal dark)
- app.js — logic prototype (Chart.js, Leaflet, controls)
- README.md — file này

Yêu cầu
- Kết nối Internet để load Google Fonts, Chart.js và Leaflet tiles (CartoDB dark basemap).
- Lưu tất cả file cùng thư mục, mở index.html bằng trình duyệt.

Tính năng mô phỏng
- Dashboard: KPI, line chart (teal + orange), donut chart, mini map, recent activity
- Shipments list: filter theo trạng thái, tìm kiếm, export CSV, nút Details
- Map view: dark basemap + route polyline và markers
- Pipeline: kéo thả deals giữa cột
- Slide-over details cho shipments
